# **App Name**: Clara.ai

## Core Features:

- AI Video Interview Analysis: Analyze pre-recorded video interviews to extract soft skills data and generate a matching score, serving as a tool for recruiters.
- Candidate Profile Management: Allow candidates to upload their CV/profile and record a video interview.
- Job Offer Creation and Management: Enable recruiters to post job offers with required hard and soft skills, facilitating the creation of tag for the skills
- Matching Score Display: Present a matching score between candidates and job offers based on AI analysis.
- Real-time Status Notifications: Provide real-time updates on application statuses through Firebase.
- User Authentication and Account Management: Implement secure user registration, login/logout, and profile management functionalities.
- Data storage: Utilize PostgreSQL and Firebase to store the user data and all related information about the recruiting process

## Style Guidelines:

- Primary color: Neon blue (#13E2FF), evoking technology and modernity.
- Background color: Dark gray (#23272A) with a desaturated blue hue, ensuring a comfortable dark theme.
- Accent color: Electric violet (#D44DFF), adding contrast and highlighting interactive elements.
- Headline font: 'Orbitron' (sans-serif) for a tech-centric feel. Note: currently only Google Fonts are supported.
- Body font: 'Poppins' (sans-serif) for a modern and readable text. Note: currently only Google Fonts are supported.
- Use neon-style icons for key functionalities and data visualizations.
- Cards with semi-transparent backgrounds and neon outlines.
- Subtle 'glow' effect on interactive buttons and elements to guide the user.
